# 🧠 SPRINT1-DASA

## 🏗️ Arquitetura da Solução

A solução proposta transforma relatórios em PDF em informações acessíveis utilizando Inteligência Artificial.

### 🔄 Pipeline Geral

Usuário ↓ Upload de PDF (Front-end) ↓ API Backend ↓ Extração de Texto (PDF/OCR) ↓ Estruturação em JSON ↓ Armazenamento ↓ Embeddings ↓ Vector Database ↓ LLM + RAG ↓ Resposta Inteligente ↓ Interface (Chat + Dashboard)

---

## ⚙️ Explicação do Pipeline

### Upload
O usuário envia o PDF através da interface.

### Processamento
O backend realiza a extração de texto do documento.

### Estruturação
Os dados são organizados em formato JSON.

### IA
A IA interpreta os dados e gera respostas.

### Interface
O usuário interage via chat e dashboard.
